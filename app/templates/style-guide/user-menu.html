We use Bootstrap's modal plugin to handle display of the user menu.

There are lots of edge cases that Bootstrap's modal plug-in handles for us, such as:

  • Ensuring that the keyboard can't be used to focus on an input "behind" the modal.
  • Ensuring that the content behind the modal isn't scrollable.
  • Ensuring that the content within the modal is scrollable.
  • Ensuring that clicking/tapping outside the modal area dismisses the modal.
  • Ensuring the ESC key can be used to dismiss the modal.
